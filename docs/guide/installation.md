# Installation

Install the package into your application using Composer:

```bash
composer config minimum-stability rc
composer require cloudcreativity/ddd-modules:^2.0
```

## Versions

| DDD Modules | PHP    |
|-------------|--------|
| `2.x`       | `^8.1` |
| `1.x`       | `^8.1` |
